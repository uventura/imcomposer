## Development

### Build
* Windows:
```
bazel build --config=imcomposer-windows imcomposer
```

* Linux:
```
bazel build --config=imcomposer-linux imcomposer
```

### Run
* Windows:
```
bazel run --config=imcomposer-windows imcomposer
```

* Linux:
```
bazel run --config=imcomposer-linux imcomposer
```
